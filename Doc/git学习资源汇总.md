# git学习资源汇总

> 本文档可以收录各种git以及团队合作规范等文章攻略的收录地，希望未来可以构建全方位的、符合规范的知识库。供后人学习……
>
> 同时汇总资源的同时也需要进行git操作，也是变向对git进行练习



1. 关于git安装与基本操作：https://blog.csdn.net/m0_73745224/article/details/149708684

> 老登批注: 大多数编辑器存在图形化操作界面，可以无需记住所有的git操作指令。但是建议了解每种指令的意义，否则会出现灾难

2. github pull request学习：[(85 封私信) GitHub pull request入门（图解+原理+git命令+可能有用的经验） - 知乎](https://zhuanlan.zhihu.com/p/672447698)

> 老登批注：pull request 是在没有加入目标仓库时，想要提交自己代码到仓库的手段。若加入目标仓库则不需要进行pull request，但是这样的操作由于没有代码审查，容易出现破坏分支的情况，慎重。

3. git 指令学习小游戏：[Learn Git Branching](https://learngitbranching.js.org/)
